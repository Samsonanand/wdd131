Site Plan - Vandaloor Zoo

Site Name

Vandaloor Zoo

The name represents one of India's largest zoological parks, dedicated to wildlife conservation and education.

Site Purpose

This website will serve as an information hub for visitors, providing details on exhibits, ticket pricing, weather conditions, conservation efforts, and upcoming events.

Target Audience

The site is intended for families, students, wildlife enthusiasts, and tourists who want to learn about the zoo before visiting.

Scenarios

  • What are the zoo's opening hours and ticket prices?
  • How can I participate in the zoo's conservation programs?
  • Which animal exhibits are available?
  • Does the zoo have any special events or guided tours?

Pages & Content

  • Home Page: Introduction, hero image, quick facts, and weather information.
  • Exhibits Page: List of animal species, habitats, and conservation programs.
  • Visitor Info Page: Ticket prices, opening hours, contact form, FAQs, and weather updates.

Navigation Structure

A common header and footer with links to:

  • Home
  • Exhibits
  • Visitor Info
  • Contact

Color Schema

Primary Color: Green (#4CAF50) – Used for headings and navigation.

Secondary Color: Beige (#F5F5DC) – Used for background and content areas.

Typography

Heading Font: Montserrat, sans-serif

Body Font: Roboto, sans-serif

Dynamic JavaScript Features

  • DOM Interaction – Update exhibit descriptions dynamically.
  • Event Listeners – Handle user interactions like button clicks.
  • Conditional Branching – Display alerts based on visitor preferences.
  • Objects & Arrays – Store and retrieve animal details dynamically.
  • Template Literals – Use dynamic strings for rendering content.
  • LocalStorage – Save user preferences or last visited exhibit.

HTML Form

The contact form will include:

  • Name
  • Email
  • Message
  • Submit button
